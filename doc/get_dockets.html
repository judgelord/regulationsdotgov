Get dockets from an agency

  • get_dockets("[agency_acronym]") retrieves dockets for an agency (see official acronyms on regulations.gov)
# you need an API key
source("~/api-key.R")

library(regulationsdotgov)

devtools::load_all()
# get FBI dockets
FBI_dockets <- get_dockets("FBI")
